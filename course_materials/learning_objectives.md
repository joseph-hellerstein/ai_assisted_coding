# Course Learning Objectives for AI Assisted Python Programming

By the end of this course, you will be able to:

1. Conduct a code review using AI assists.
2. Know good prompting skills.
3. Implement a small feature using AI assists.
4. Build unit tests using AI assists.
5. Write a large feature using AI assists.
6. Use subagents and skills to improve AI assists for code review, feature implementation and unit tests.
7. Use MCP tools to test your application.
