# Audience

Attendees will be academics who want to improve their productivity with building applications in Python. Attendees should
have substantive experience with Python programming and ``github`` with some exposure to AI chatbots. Their skills include:

* They have written python applications in excess of 1,000 lines of code.
* They have written applications with at least two Python modules.
* They have at least one ``github`` repository in which they are the primary contributor.
* They have developed an application in ``VSCode``.
* They have used a large language model to write code.
* They have written basic HTML tags.
* Ideally, they have created a PyPI package.

Attendees are expected to complete the following prior to the start of the course:

1. Students have a laptop.
2. Student has VSCode installed on their computer.
3. Create a ``github`` repository with the title ``ai_assisted_python_programming``.
4. Install Dash on your laptop.
5. Write a Dash application that creates a web page that says "Hello World!". All the files for the application should be in your github repository.
